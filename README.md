# rpls
tidal repitch echo /
phonetic fracture prism

- **K2-K3**: page
- **E1-E3**: various
- **K1**: alt

time flows thru 3 pools
the first reads, 
the others write.
set the rate & size of 
these pools to draw
concentric prisms.

( ok it's a bit like a multi-tap 
delay, but each tap plays 
back at a different rate )
